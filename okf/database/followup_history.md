# Followup History Table

## Columns

| Column | Type | Description |
|---|---|---|
| `id` | INTEGER (PK) | Auto-increment |
| `task_id` | TEXT (FK → tasks.id ON DELETE CASCADE) | Owning task |
| `message_text` | TEXT | The followup message text sent |
| `recipient_jid` | TEXT | The JID the nudge was sent to (group or individual) |
| `sent_at` | TIMESTAMP | When the message was sent (auto-set) |

## Purpose

Tracks when followup nudges were sent for each task. Nudges are generated by `POST /api/nudge/generate` (Gemini generates 3 variants: gentle, passive-aggressive, aggressive) and then sent via the v1 `POST /api/send` endpoint.
