# Gemini Prompts

- [Discover Tasks Prompt](discover_tasks.md) — Extracts tasks from 30-day message transcripts. Model: gemini-3.1-flash-lite-preview.
- [Refresh Tasks Prompt](refresh_tasks.md) — Incrementally updates existing tasks with new messages. Per-task status, progress, and importance.
- [Deep-Dive Task Prompt](deep_dive_task.md) — Generates a comprehensive knowledge page for a single task. Full transcript analysis.
